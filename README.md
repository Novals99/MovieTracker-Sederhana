# 🎬 Movie Tracker

Movie Tracker adalah aplikasi web sederhana berbasis Flask yang digunakan untuk mengelola koleksi film.

## Fitur

- Menampilkan daftar film
- Menambahkan film baru
- Mengubah data film
- Menghapus film
- REST API menggunakan Flask
- Database MariaDB
- Reverse Proxy menggunakan Nginx
- Cloudflare Tunnel
- Deploy menggunakan Proxmox VE dan LXC
